Responsive Font Strategies

em's

  • A much more flexible way of sizing fonts is to use ems. One em is equal to the current font size. If the body text is 16px then 1em = 16px; 2em = 32px;

  • Ems are resizable across browsers (zoom friendly).

  • Ems cascade - that is, the font size of the parent element effects the font size of the child element.

  • Quiz: Given the following CSS, what is the font-size of the span element below?

      body {
        font: normal 100%/1.5em Helvetica, sans-serif; /* font-size: ~16px, line-height: ~24px; */
      }
      p {
        font-size: 1.25em; /* 20px / 16px */
      }
      span {
        font-size: 1em; /* 16px / 16px */
      }

<p><span>This is an example of the cascading issue when using ems.</span> This span element's font-size is set to 1 em in CSS. And its actual size is calculated as 1em of its parent p tag resulting in a size of 20px. With rems, we can avoid cascading issues that occur because rems (root ems) calculate their base on the root html element.</p>

TIP: When using ems it is recommended to set the base font in percentages to avoid a resizing issue that Internet Explorer has. IE's resizing for differing em sizing isn't consistent with other browsers.
