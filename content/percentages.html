Responsive Font Strategies

Percentages

  • Font sizes in percentages are similar to ems. They are resizable and they cascade - that is, the font size of the parent element effects the font size of the child element. If the base font-size is 16px then 100% = 16px; 200% = 32px;

  • NOTE: There isn't any major difference between ems vs percentages except that ems are often the preferred choice.
