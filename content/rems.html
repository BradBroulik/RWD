Responsive Font Strategies

rem's

  • Rems (root ems) behave similar to ems with one important difference: rems size relative to the root html element and not the size of the body element.

  • Since rems size relative to the root html element they do not cascade - that is, the font size of the parent element does not effects the font size of the child element.

  • Quiz: Given the following CSS, what is the font-size of the span element below?

      html {
        font: normal 100%/1.5rem Helvetica, sans-serif; /* font-size: ~16px, line-height: ~24px; */
      }
      p {
        font-size: 1.25rem; /* 20px / 16px */
      }
      span {
        font-size: 1rem; /* 16px / 16px */
      }

  • <p><span>With rems, we can avoid cascading issues that occur with nested elements.</span> As expected, the size of this span element is 1rem (16px).</p>

  • Browser support for rems is good except for some legacy browsers and Opera mobile. For fallback support you can provide a pixel fallback: span { font-size: 16px; font-size: 1rem; }
