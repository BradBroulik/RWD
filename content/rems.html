Responsive Font Strategies

rem's

  • Rems (root ems) behave similar to ems with one important difference: rems size relative to the root html element and not the size of the body element.

  • <p><span>With rems, we can avoid cascading issues that occur with nested elements. As expected, the size of this span element is 1rem (16px).</span></p>

  • Browser support for rems is good except for some legacy browsers and Opera mobile. For fallback support you can provide a pixel fallback: span { font-size: 16px; font-size: 1em; }
